# 1IN1B Agency

> **We turn ideas into code.** — Bibhuti Hai to munkin Hai

1IN1B is a highly specialized, agile software development studio. We build scalable systems like POS, CRMs, ATS, Ticketing, and specialized dashboard solutions, alongside robust desktop and mobile apps. We are the engineering layer for your business.

## Our Process

```
01 ── Discovery & Architecture
      Map your ideas, understand constraints, and architect a minimal,
      scalable foundation. Tech stack, database schema, and core UI
      paths are defined here.

02 ── Intuitive UI/UX Prototyping
      Design high-fidelity, frictionless interfaces focused strictly
      on utility, stripping away unnecessary visual noise.

03 ── Agile Engineering
      Build in iterative cycles with frequent progress sharing.
      Code is modular, type-safe, and thoroughly tested.

04 ── Deployment & Scale
      Secure infrastructure and CI/CD pipelines. Post-launch
      monitoring ensures smooth scaling as your user base grows.
```

## Featured Work

### UfdLoader
Terminal-based download accelerator — multi-connection segmented downloads, resume support, cross-platform native binaries, real-time terminal UI with per-connection progress bars.
- [Source](https://github.com/1IN1B/ufd) · [Live Demo](https://1in1b.github.io/ufd/)

### Xplore
Minimalist anonymous video and text chat application — an Omegle-inspired clone built with Next.js, WebRTC, and Socket.io. Features P2P video/audio streaming, real-time messaging, skip-to-next-partner.
- [Source](https://github.com/1IN1B/xplore) · [Live Demo](https://xplore-production.up.railway.app/)

### TicketGo
Modern ticket management system with authentication, role-based access, form validation, and database integration. Built with Next.js, TypeScript, MySQL, NextAuth, Radix UI, and Zustand.
- [Source](https://github.com/1IN1B/ticketgo) · [Live Demo](https://ticketgo-wine.vercel.app/)

## Getting Started

```bash
# Install dependencies
npm install

# Set your Gemini API key
# Create .env.local with:
#   GEMINI_API_KEY=your_key_here

# Start development server
npm run dev
```

**Scripts:**

| Command | Description |
|---|---|
| `npm run dev` | Start dev server on port 3000 |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | TypeScript type-check |
| `npm run clean` | Remove dist and server artifacts |
